Lista ordenada de pasos para un algoritmo cualquiera que conidere condiciones y repeticiones

  1. Iniciar sesión en la plataforma
  2. Seleccionar el curso deseado
  3. Verificar si hay nuevas lecciones disponibles
    • Sí: Continuar con la lección
    • No: Esperar hasta que haya nuevas lecciones
  4. Completar las actividades y ejercicios
    • Si se completa correctamente, avanzar a la siguiente lección
    • Si no se completa correctamente, revisar el material y volver a intentar
  5. Finalizar el curso y obtener el certificado
